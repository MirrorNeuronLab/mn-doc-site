# Install MirrorNeuron Locally

This guide installs the dependencies needed to validate bundles, run local workflows, and execute the test suite.

## Requirements

- macOS, Linux, or WSL2
- Docker

Docker Desktop includes the plugin on macOS and Windows. On Linux, please install the runtime plugin manually:

```bash
sudo apt-get update
sudo apt-get install docker-model-plugin
```


## Option 1: Install With The Deployment Script

Use the deployment script when you want a system-wide `mn` command.

```bash
curl -fsSL https://mirrorneuron.io/install.sh | bash
```
 
or you get get install.sh and install it manually with options:

```text
wget https://mirrorneuron.io/install.sh
bash install.sh --help
bash install.sh
```

Verify:

```bash
mn --help
```

This will install our latest released verison. If you are looking for legacy versions, please check the github releases.


## Configure Local Environment

most time, user do not need to configure local environment variables, but if you want to change the default install path, using dev mode. please read ([Development Guide](development.md)).

## Uninstall MirrorNeuron
Stop MirrorNeuron:

```bash
mn runtime stop
```

```bash
curl -fsSL https://mirrorneuron.io/uninstall.sh | bash
```

Remove all data and configuration files:
```bash
rm -f ~/mn
```

If you installed on a cluster, run above command on all nodes to remove MirrorNeuron.
