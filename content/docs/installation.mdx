# Install MirrorNeuron

This guide installs the dependencies required to validate bundles, run local workflows, and execute the test suite.

## Requirements

- macOS, Linux, or Windows with WSL2
- Docker

Docker Desktop includes the required plugin on macOS and Windows. On Linux, install the runtime plugin manually:

```bash
sudo apt-get update
sudo apt-get install docker-model-plugin
```

## Install with All-in-one Deployment Script

Use the deployment script when you want to install MirrorNeuron system-wide and make the `mn` command available from your shell.

```bash
curl -fsSL https://mirrorneuron.io/install.sh | bash
```

You can also download `install.sh` and run it manually with options:

```text
wget https://mirrorneuron.io/install.sh
bash install.sh --help
bash install.sh
```

Verify the installation:

```bash
mn --help
```

This installs the latest released version of MirrorNeuron. If you need a legacy version, check the GitHub releases.

## Configure the Local Environment

Most users do not need to configure local environment variables. If you want to change the default installation path or use development mode, see the [Development Guide](development.md).

## Uninstall MirrorNeuron

Stop MirrorNeuron:

```bash
mn runtime stop
```

Run the uninstall script:

```bash
curl -fsSL https://mirrorneuron.io/uninstall.sh | bash
```

Remove all data and configuration files:

```bash
rm -f ~/mn
```

If MirrorNeuron is installed on a cluster, run the commands above on every node to remove it completely.
