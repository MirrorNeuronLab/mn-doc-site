# Install MirrorNeuron

Use the deployment installer for a supported local runtime. Use local mode when you are working from the MirrorNeuron workspace and need editable components.

## Requirements

- macOS, Linux, or Windows with WSL2
- Docker running locally
- `git`

For local development, also install Python 3.11+ and Elixir/Erlang. Docker Desktop includes Docker Model Runner on macOS and Windows; on Linux, install the Docker Model Runner plugin separately if you plan to use local models.

## Install a Released Runtime

From a checkout of the deployment repository:

```bash
cd mn-deploy
./install.sh --help
./install.sh
```

Or use the hosted installer:

```bash
curl -fsSL https://mirrorneuron.io/install.sh | bash
```

The installer is non-interactive by default. Pass `--interactive` to review its choices or `--version <tag>` to install a matching released component set.

Verify the CLI and start the services:

```bash
mn --help
mn runtime start
mn runtime health
```

The deployment scripts store local runtime state under `~/.mn` by default. The REST API listens on `http://localhost:54001/api/v1` unless `MN_API_PORT` is set.

## Install From a Workspace Checkout

From the workspace root, run the installer in local mode:

```bash
cd mn-deploy
./install.sh --mode local
```

This is the supported route for an editable local installation. After it completes:

```bash
mn runtime start
mn node list
```

If you are changing the BEAM core directly, fetch and compile its dependencies from the core repository:

```bash
cd MirrorNeuron
mix deps.get
mix compile
```

## Optional: Local Model Runtime

For blueprints that use Docker Model Runner, inspect and install the requested model before launching the workflow:

```bash
mn model list
mn model install gemma4:e2b
mn model doctor gemma4:e2b
```

Read [Model Runtime](model-runtime.md) for model selection and remote-provider configuration.

## Validate the Setup

Validate a checked-in blueprint before submitting it:

```bash
mn blueprint validate otterdesk-blueprints/generic_customer_service_voice_coworker
```

Validation checks the bundle and its declared requirements without starting a job. Continue with [Quickstart](quickstart.md) when it succeeds.

## Uninstall

Stop the local runtime first:

```bash
mn runtime stop
```

If you installed with the hosted installer, run its matching uninstall script:

```bash
curl -fsSL https://mirrorneuron.io/uninstall.sh | bash
```

Only remove `~/.mn` manually when you intend to delete its local runtime configuration, run records, and model metadata.
